IU880B-USB

    .NET Library for communicating with the WiMOD iU880B USB (and iM880B) LoRaWAN Modem.

    Features:

    • 1.23 HCI Spec
    • .NET Standard

    Documentation

    Documentation available here

    Getting Started.

    1. Check out the repo or reference from nuget by typing in your Package Manager Console

      Install-Package FosterBuster.IU880B -Version 1.0.0

    2. Have a look at the example project for a quick example of how to get rolling with activating the device, sending and receiving messages and so forth. If you have any questions or issues, please don't hesitate to ask.
