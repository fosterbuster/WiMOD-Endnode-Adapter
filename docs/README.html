IU880B-USB

    .NET Library for communicating with the WiMOD iU880B USB (and iM880B) LoRaWAN Modem.

    An adapter (or whatever you could call it) for communicating with IU880B LoRaWAN EndNode Modems (tested on the iU880B). Most core functionality is implemented.

    MIT but if you improve upon it, please do a pull-request. And if you use it, i'd love to know!

    Example:

    To get rolling, ensure that the device is configured by sending a NetworkStatusRequest. If the device is not active, you have two options: ABP or OTAA. (relevant classes here)

    To configure for OTAA, first send a SetJoinParameters request, when the device says OK. Send a OTAA reqest. For ABP, just send an ABP message.

    Wait for a Join Network Indication. When you get one, you are ready to send messages uplink.

    Send a Reliable Data Message (requires ACK, so you shouldn't really use it, think of the gateways dutycycle), or an Unreliable Data Message (fire and forget).
